# Project 4 – Data Visualization

## Overview

This project focuses on visualizing sales data using charts and graphs to better understand product performance and business trends.

## Tools Used

* Microsoft Excel
* Charts and Graphs

## Visualizations Created

### Chart 1 – Total Orders by Product

Shows the number of orders received for each product.

**Insight:**

* Printer received the highest number of orders (181).
* Phone received the lowest number of orders (156).

### Chart 2 – Total Sales by Product

Displays total sales generated by each product.

**Insight:**

* Printer generated the highest total sales (195,612.61).
* Phone generated the lowest total sales (151,722.39).

### Chart 3 – Average Sales by Product

Shows average sales value for each product.

**Insight:**

* Laptop had the highest average sales value (1110.56).
* Phone had the lowest average sales value (972.58).
